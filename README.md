# F2コンテンツ日本語化プロジェクト

FinOps Foundation（F2）コンテンツの日本語化プロジェクトが管理するリポジトリで、日本語済みコンテンツを公開するウェブサイトの機能も有しています。

日本語化ドキュメントはMarkdownで管理、ウェブサイトは[Docusaurus](https://docusaurus.io/)を使って構築してます。
